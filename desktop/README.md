# Proxy Access Desktop

Десктопное приложение для подключения к прокси-серверу.

## Установка

```bash
cd desktop
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Использование

1. **Ключ активации** — вставьте ключ, выданный после регистрации (кнопка "Обновить ключ" в личном кабинете).
2. **Bearer токен** — вставьте `access_token` из `localStorage` браузера (нужен только для отключения).
   Открыть: DevTools → Application → Local Storage → `access_token`.
3. Нажмите **Подключиться** — приложение покажет хост, порт и протокол выданного прокси.
4. Статус обновляется автоматически через WebSocket.
5. Нажмите **Отключиться** для освобождения прокси.

## Статусы

| Статус | Цвет | Описание |
|---|---|---|
| Подключено | Зелёный | Прокси назначен и активен |
| Отключено | Серый | Нет активного подключения |
| Нет свободных прокси | Красный | Все VM заняты, попробуйте позже |
| Ошибка | Красный | Ошибка сети или сервера |

## Требования

- Python 3.11+
- Backend запущен на `http://localhost:8002`
